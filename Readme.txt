Logs and displays visitor IP addresses for website security and monitoring.

== Description ==

IP Informant Logger is a WordPress plugin that logs and displays visitor IP addresses. This tool is essential for website administrators who wish to monitor their site's traffic and enhance security measures.

Key Features:
- Log visitor IPs
- Display logged IP addresses in the WordPress admin
- Integration with IPinfo.io for detailed IP information

== Installation ==

1. Upload the `ip-informant-logger` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings page to configure the IPinfo.io API key.

== Frequently Asked Questions ==

= Where do I get an IPinfo.io API key? =

You can obtain an API key by signing up at https://ipinfo.io/.

= Is this plugin free to use? =

Yes, IP Informant Logger is free to use, but utilizing IPinfo.io services may require a subscription for extensive usage.
